A real-time multiplayer math battle game where two players race to solve math questions as quickly and accurately as possible.

mathduel-preview.png


## ✨ Features

- ⚔️ Real-time 1v1 multiplayer gameplay
- 🧮 Randomly generated math questions
- ⚡ Live score updates using Socket.IO
- 🏆 Winner detection
- 📱 Responsive interface
- 🎨 Clean modern UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Socket.IO
- SQLite

## 🚀 Live Demo

https://math-duel-production.up.railway.app/

## 💻 Run Locally

```bash
git clone https://github.com/aaditdahiya/Math-Duel.git
cd Math-Duel
npm install
npm start
```

Then open:

```
http://localhost:3000
```
